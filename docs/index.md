# PulseAI Platform Documentation

Welcome to the PulseAI knowledge base! This documentation platform is designed to be seamless, beautiful, and completely Markdown-driven.

## Features

- **Markdown-Driven**: Just drop `.md` files in the `docs/` folder to create a new page.
- **Syntax Highlighting**: Built-in support for multiple languages, powered by Highlight.js.
- **Auto-Generated Navigation**: All files automatically appear in the left sidebar, formatted cleanly.
- **Auto-Generated TOC**: The platform parses your headings (`h2`, `h3`) and generates a table of contents on the right side.
- **Dark Mode Native**: Tailwind typography and custom styles ensure deep integration with the platform's theme.
